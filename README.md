# Sudoku Game AI

A Python-based Sudoku game and puzzle management project developed using Object-Oriented Programming principles.  
The project includes Sudoku board handling, puzzle loading, game logic, and field management systems for interactive Sudoku gameplay and problem solving.

## Features

- Sudoku puzzle loading
- Board and field management
- Interactive game logic
- Puzzle validation
- Multiple Sudoku test boards
- Object-oriented system design

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- Game Logic
- File Handling

## Main Components

Sudoku

Handles Sudoku board representation and puzzle-related operations.

Field

Represents individual Sudoku cells and value management.

Game

Controls gameplay flow, puzzle interaction, and validation logic.

App

Main entry point for running the Sudoku application.

## Concepts Demonstrated

* Object-oriented programming
* Board state management
* Puzzle validation
* File-based puzzle loading
* Interactive game systems

## Purpose

This project was developed to explore Sudoku game implementation, puzzle management, and object-oriented software design through Python.
